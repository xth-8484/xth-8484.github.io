Typora Markdown Grammar

title 标题样式

​ 一级标题 - ctrl + 1
​ 二级标题 - ctrl + 2
​ 三级标题 - ctrl + 3
​ 四级标题 - ctrl + 4
​ 五级标题 - ctrl + 5
​ 六级标题 - ctrl + 6

font 文本样式

​ 加粗 - Ctrl + B，前后加两个 *
​ 斜体 - Ctrl + I，前后加一个 *
​ 下划线 - Ctrl + U，或者使用 <u> </u>
​ 高亮 - 前后加两个 =
​ 删除线 - 前后加两个 ~
​ 上标 - 使用 ^ ^
​ 下标 - 使用 ~ ~
​ 注释 - 使用 <!– –>

blockquote 块

​ 插入表格 - Ctrl + T
​ 插入代码块 - Ctrl + Shift + K 或 ```
​ 插入公式块 - Ctrl + Shift + M
​ 引用 - Ctrl + Shift + Q 或 > + Space
​ 有序列表 - Ctrl + Shift + [ 或 1. + Enter
​ 无序列表 - Ctrl + Shift + ] 或 - + Space
​ 数学表达式 - $ $

link 链接

​ 超链接 - 使用 [ ]( )
​ 图像链接 - Ctrl + Shift + I
​ Emoji 表情 - : :

break 换行

​ 软换行 - Shift + Enter
​ 硬换行 - Space Space + Shift + Enter
​ 换段 - Enter 或两次软换行
